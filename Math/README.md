### Targets
- net8.0
- net9.0

### Classes, Records & Structs

#### Interval\<T\>
Implements IInterval.

#### IIntervalExtensions
Extends IInterval to include GetFirstOf, GetLastOf and GetEnumerator; for byte,
sbyte, short, ushort, int, uint, long and ulong.

#### Stats
A set of static statistics algorithms.
